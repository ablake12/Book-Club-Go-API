Hello! Welcome to the Book Club API. This is where you'll be able to manage your book club and make your API calls.

Your options to use the API are:

  • Through the interface below
  • Through Postman

Enjoy!


GET
GET All Books   /books
GET Read Books   /books/read
GET Unread Books   /books/unread
GET Current Book   /books/current
GET Reviews by Book ID   /books/<book_id>/reviews


POST
POST a Book   /books












POST a Review by Book ID   /books/<book_id>/reviews








PUT
PUT a Book Read Status by Book ID   /books/<book_id>/status


PUT a Book Current Status by Book ID   /books/<book_id>/current


PUT a Review by Book and Reviews IDs   /books/<book_id>/reviews/<review_id>








DELETE
DELETE a Book by Book ID   /books/<book_id>


DELETE a Review by Book and Review IDs   /books/<book_id>/reviews/<review_id>




DELETE All Books   /books
